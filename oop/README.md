# オブジェクト指向練習問題

Rubyでオブジェクト指向プログラミングを身につけるための練習問題集。

## 目的

- オブジェクト指向の基本概念を**手を動かしながら**理解する
- 「なぜそう書くのか」を**言語化できる**ようになる
- 将来的にRailsで設計判断ができる土台を作る

## ディレクトリ構成

```
oop/
├── README.md           # このファイル
├── exercises/          # 練習問題
│   ├── 01_class_and_instance/
│   │   ├── README.md   # 問題文
│   │   ├── counter.rb  # 解答（学習者が作成）
│   │   └── REVIEW.md   # レビュー（AI が作成）
│   ├── 02_encapsulation/
│   └── ...
└── spec/               # テスト
    ├── spec_helper.rb
    └── 01_class_and_instance/
        └── counter_spec.rb
```

## 学習の進め方

### 1. 問題を読む

`exercises/XX_xxx/README.md` を読む。問題文だけでなく、**考えてほしいこと**のセクションにある問いについてまず考える。

### 2. 実装する

- 指定されたファイル（例: `counter.rb`）を作成
- テストを実行して動作確認（`rspec spec/XX_xxx/`）

### 3. プッシュして伝える

実装が終わったら、コミット&プッシュして「書いたよ」とAIに伝える。

### 4. 質疑応答

AIがコードを読んで質問する。**なぜそう書いたか**を説明する。この対話を通じて理解を深める。

### 5. レビューの記録

レッスン終了時、AIが `REVIEW.md` を作成する。学習した概念、良かった点、次に学ぶべきことがまとまる。

### 6. 次の問題へ

理解が確認できたら、次の練習問題に進む。
