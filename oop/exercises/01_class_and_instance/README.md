# 01: クラスとインスタンス

## 題材：カウンター

ボタンを押すと数が増えるカウンターを作ってください。

### 要件

- カウンターは最初 **0** からスタートする
- `click` すると数が **1** 増える
- `count` で現在の数を取得できる

### 考えてほしいこと

この問題を解く前に、以下の問いについて考えてみてください：

1. **「状態を持つ」とはどういうことか？**
   - カウンターが「今いくつか」を覚えているのは、どこに保存されている？

2. **なぜクラスを使うのか？**
   - もしクラスを使わずに実装するとしたら、どうなる？
   - その場合、何が困る？

3. **インスタンスとは何か？**
   - `Counter.new` を2回呼んだら、何が起きる？
   - それぞれのカウンターは独立している？共有している？

### ファイル

- `counter.rb` を作成してください
- テストは `spec/01_class_and_instance/counter_spec.rb` にあります

### テストの実行

```bash
cd oop
rspec spec/01_class_and_instance/counter_spec.rb
```

### キーワード

調べてみよう：
- インスタンス変数 (`@variable`)
- `initialize` メソッド
- カプセル化（なぜ `@count` を直接触らせないのか）
