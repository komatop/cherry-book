# Web エンジニアリング基礎

コードを書かずに、Webの仕組みを深く理解するための学習コンテンツ。

## 学習形式

1. **レッスンを読む** - 概念と「なぜそうなっているか」
2. **質問に答える** - 選択式・記述式の問題
3. **質疑応答** - AIとの対話で理解を深める
